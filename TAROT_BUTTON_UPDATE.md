## 🎨 设计特点

### 配色方案

**Hero 按钮（顶部）**:
- 主色：紫色 (`purple-600`) → 粉色 (`pink-600`) → 靛蓝 (`indigo-600`)
- 悬停：深紫色 (`purple-700`) → 深粉色 (`pink-700`) → 深靛蓝 (`indigo-700`)
- 装饰：金色图标 (`gold-400`)

**CTA 按钮（底部）**:
- 主色：金色 (`gold-500`) → 橙色 (`orange-500`) → 金色 (`gold-600`)
- 悬停：深金色 (`gold-600`) → 深橙色 (`orange-600`) → 深金色 (`gold-700`)
- 文字：黑色（与金色背景形成强烈对比）

### 尺寸规格

| 位置 | 文字大小 | 内边距 | 圆角 |
|------|----------|--------|------|
| Hero | 2xl-3xl | 12×8 | 2xl (rounded-2xl) |
| CTA  | 3xl-4xl | 16×8 | 2xl (rounded-2xl) |

### 响应式设计

```css
/* 移动端 */
- 文字：text-2xl (Hero), text-3xl (CTA)
- 保持完整功能和动画

/* 桌面端 */
- 文字：text-3xl (Hero), text-4xl (CTA)
- 更大的内边距和更明显的效果
```
