# 直接访问对话式占卜

## 🎯 更新说明

现在点击"开始占卜"按钮会**直接进入对话式AI占卜界面**，不再需要经过模式选择页面！

## ✨ 新的访问方式

### 1. 直接对话式占卜（推荐）

**按钮**: 主页的"开始占卜"按钮  
**链接**: `/tarot?mode=chat`  
**效果**: 直接进入与AI塔罗师的对话界面

### 2. 快速占卜模式

**链接**: `/tarot?mode=quick`  
**效果**: 直接进入传统的表单式占卜界面

### 3. 模式选择页面

**链接**: `/tarot`（不带参数）  
**效果**: 显示两种模式的选择界面

## 🎨 实现方式

### URL参数自动识别

```tsx
// 组件会自动检测URL参数
useEffect(() => {
  const mode = searchParams.get("mode");
  if (mode === "chat") {
    setSelectedMode("chat");  // 自动进入对话模式
  } else if (mode === "quick") {
    setSelectedMode("quick"); // 自动进入快速模式
  }
}, [searchParams]);
```

## 🎯 设计理念

### 为什么直接进入对话模式？

1. **最佳体验**: 对话式占卜是最创新、最有特色的功能
2. **用户友好**: 新用户不需要理解两种模式的区别
3. **引导明确**: AI会自动引导用户完成整个流程
4. **降低门槛**: 减少决策步骤，提高转化率

### 快速模式的访问

虽然主按钮进入对话模式，但快速模式仍然可以通过：
- 直接访问 `/tarot?mode=quick`
- 在 `/tarot` 页面选择
- 在对话界面点击"返回选择模式"
